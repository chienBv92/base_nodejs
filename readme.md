# Database
### Create migration
```shell
npm run typeorm migration:create -- --name=CreateUserTable
```

### Run migration
```shell
npm run typeorm migration:run
```

### Reverts last executed migration
```shell
npm run typeorm migration:revert
```

# Run
###Install package
npm run i

### Develop
```shell
npm run dev
```
### Production
```shell
npm run prod
```

# Coding style
```shell
npm run lint:fix
```
